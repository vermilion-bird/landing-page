Codebase RAG 原理

1. 语义编码 → embedding model
  // text → vec (1536维)
2. 向量检索 → KNN Search
  // Top-K 相关代码块
3. 重排序 → Cross-Encoder
  // 精排提升精度
4. Prompt 增强 → LLM 生成
  // 构建完整上下文
索引粒度: 函数级 + 类级
Embedding: text-embedding-3-small
